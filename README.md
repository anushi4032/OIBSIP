##  Task Breakdowns

### Task 2: Guess The Number (CLI)
A robust command-line number-guessing game featuring dynamic configurations, exception handling, and custom state tracking.

*   **Key Features:**
    *   **Three Game Modes:** Easy (1-50, 10 turns), Medium (1-100, 7 turns), and Hard (1-200, 5 turns).
    *   **Telemetry Dashboard:** Tracks a rolling real-time execution log of user matches, wins, defeats, and structural turn telemetry.
    *   **Input Validation:** Fully protected against `NumberFormatException` inputs and out-of-bounds boundary selections without penalizing game health state loops.

### Task 4: MCQ Exam System (GUI)
A fully functional Swing graphical user interface implementing an automated multiple-choice examination framework.

*   **Key Features:**
    *   **Profile Control & Authentication:** Modular architecture incorporating secure conditional view swapping using a `CardLayout`.
    *   **Active Countdown Timer:** Features a 30-minute background swing execution timer with automatic forced evaluation and submission logic upon expiration.
    *   **Interactive Navigation & Summaries:** Comprehensive historical matrix detailing targeted questions, provided answers, correct benchmarks, and instant score breakdowns.
